# Class 12 - Backend Mock Start

## Run
1. `npm install`
2. `npm run start:dev`
3. Open Swagger: `http://localhost:5001/api`

## Run from VS Code (Play/F5)
- Open this folder in VS Code.
- Run configuration: `Launch Nest Application`.
- This uses `npm run start:debug` (same style as full project backend).

## Goal
Use this backend as the base to implement controller + service + module architecture step by step.
