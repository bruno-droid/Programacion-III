# Step 1 Demo - Controller + Service + Module

## Run
1. `npm install`
2. `npm run start:dev`
3. Open Swagger: `http://localhost:5001/api`

## What this demo teaches
- `DemoMessagesController`: thin HTTP layer.
- `DemoMessagesService`: in-memory business logic.
- `DemoMessagesModule`: composition of controller + service.
